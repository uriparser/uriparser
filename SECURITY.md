# Security Policy

If you have discovered and humanly validated a security vulnerability in uriparser,
unless it is a case of [arbitrary code execution](https://en.wikipedia.org/wiki/Arbitrary_code_execution)
with a [proof of concept](https://en.wikipedia.org/wiki/Proof_of_concept),
please report it as a [public issue to this repository](https://github.com/uriparser/uriparser/issues/new)
with all details necessary to understand and reproduce the issue.

In case of arbitrary code execution,
please email me at [sebastian@pipping.org](mailto:sebastian@pipping.org), instead.

Please note that uriparser is maintained by a single unpaid volunteer in his spare time.
If your business relies on uriparser
being secure,
reacting timely, and/or
keeping you in the loop:
please consider reaching out for a [support contract](mailto:sebastian@pipping.org?subject=uriparser%20support%20contract).

Thank you!

Sebastian Pipping, Berlin, 2026-06-15
